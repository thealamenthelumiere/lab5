# Отчет по лабораторной работе №4
# Написание Unit-тестов

## 1. Цели и задачи тестирования
**Цель:** Проверить корректность работы функций вычисления площади и периметра прямоугольника

**Задачи:**
- Проверить вычисление площади при различных входных данных
- Проверить вычисление периметра при различных входных данных
- Убедиться в обработке граничных случаев (нулевые значения)

## 2. Описание тестируемого продукта
Тестируются две математические функции:
- `area(a, b)` - вычисляет площадь прямоугольника
- `perimeter(a, b)` - вычисляет периметр прямоугольника

## 3. Область тестирования
Тестируются следующие компоненты:
- Функция вычисления площади
- Функция вычисления периметра
- Обработка целочисленных входных данных

## 4. Стратегия тестирования
Используется модульное тестирование (Unit testing) с помощью фреймворка unittest
Тестируются:
- Нормальные случаи (положительные числа)
- Граничные случаи (нулевые значения)
- Специальные случаи (квадрат)

## 5. Критерии приемки
- Все тесты должны выполняться без ошибок
- Код покрытия тестами - 100% для основных сценариев
- Соответствие ожидаемых и фактических результатов

## 6. Ожидаемые результаты
- Отчет о выполнении тестов
- Статус "OK" для всех тестов
- Подтверждение корректности работы функций
